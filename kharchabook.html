💾 Data Kahan Hai?

Tera data sirf is device ke browser (IndexedDB) mein save hai. Koi server nahi, koi cloud nahi. Cache clear karne se bhi IndexedDB data safe rehta hai normally. Extra safety ke liye neeche se backup le lo.

📥 Backup Download Karo

Sab transactions + photos ek .json file mein download honge. Apne Downloads folder ya Google Drive mein save karo.

📤 Backup Restore Karo

Pehle download ki hui .json backup file select karo. Data wapas aa jaayega.

⚠️ Sab Delete Karo

Saare transactions aur photos permanently delete ho jaayenge. Pehle backup zaroor le lo!
